HOW TO INSTALL:

- Download and run the setup.exe file.
- When asked, give permission to complete the program installation
- Copy the hoposim.lic license file to one of the folders suggested by the program.
- Download and unpack the HoPoSim3D.zip archives
- Start the HoPoSim application and set the 3D simulator path (HoPoSim3D.exe from the archive) under Settings/3D Simulator.
