

HoPoSim (HolzPolterSimulator) Software
Software developer: Dr. Philippe Guigue (http://philippe-guigue.de)


HOW TO INSTALL:

- Execute the file „setup.exe“.
- When prompted, grant rights to install software.
- Copy the „hoposim.lic“ license file in one of the folders, that is suggested by the program.
- Copy the „HoPoSim3D“ folder to the HoPoSim-location on your hard drive.
- Start the HoPoSim application and set the path to the 3D simulator (HoPoSim3D.exe) under „Einstellungen/3D Simulator“.
